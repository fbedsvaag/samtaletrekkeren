# BPG Samtaletrekkeren

Et enkelt verktøy for planlegging av muntlige fagsamtaler og mentorsamtaler ved Bergen Private Gymnas.

**→ [Åpne verktøyet](https://fbedsvaag.github.io/samtaletrekkeren/)**

---

## Om

BPG Samtaletrekkeren genererer en tilfeldig trekningsliste og timeplan for individuelle elevsamtaler. Du legger inn klasselisten, setter opp vurderingsdager med klokkeslett, og appen fordeler elevene automatisk.

Resultatet kan eksporteres direkte til Teams, skrives ut som PDF, eller kopieres som Markdown.

---

## Funksjoner

- **Fagsamtale eller mentorsamtale** — bryter øverst tilpasser visning og eksport
- **Klasseliste** — lim inn navn eller last opp CSV-fil
- **Oppgavenummer og tema** — opptil 8 oppgaver med egne navn og farger
- **Vurderingsdager** — flere dager, valgfri lunsjpause per dag
- **Trekning med låser** — lås tema, rekkefølge, eller individuelle elever mellom trekkinger
- **Drag-and-drop** — flytt elever manuelt i timeplanen etter trekking
- **Tre visninger** — Tema-fordeling · Timeplan · Alfabetisk liste
- **Eksport** — Kopier til Teams (HTML-tabell) · Kopier som Markdown · Skriv ut / PDF
- **Utskrift** — velg hvilke seksjoner som skrives ut, med BPG-logo på hver side

---

## Teknisk

Verktøyet er én enkelt selvinneholdt `.html`-fil — ingen server, ingen installasjon, ingen innlogging. Fungerer lokalt i alle nettlesere; Chrome eller Edge anbefales for Teams-eksport (krever Clipboard API).

---

## Oppdatere verktøyet

Når en ny versjon av `index.html` er klar:

1. Gå til dette repositoryet på GitHub
2. Klikk **Add file → Upload files**
3. Last opp den nye `index.html`
4. Klikk **Commit changes**

GitHub Pages oppdaterer seg automatisk innen 1–2 minutter.
